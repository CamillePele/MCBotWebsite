# MCBot Website
@Deprecated

MCBotWebsite is website that allow you to manage minecraft afk bot to load easily chunck on your minecraft server

  - Connect/Disconnect player
  - Add easily an account
  - Low security system

### Tech

MCBot Website uses a number of open source projects to work properly:

* [Express] - To do the backend
* [Minecraft Protocol] - handle minecraft account
* [node.js] - evented I/O for the backend
